Employee performance prediction

This is a machine learning model that predicts the performance of an employee based on his/her past performance. The model is trained on a dataset of 1200 employees and their performance. The model is trained using a neural network with 3 hidden layers. The model is trained on 80% of the dataset and tested on the remaining 20% of the dataset. The model has an accuracy of 98%.
